JavaScript "+" 运算符

JavaScript 数据类型：

  • 基本数据类型：Null , Undefined, Number, String, Boolean, Sysbol(ES6);
  • 复杂数据类型：Object

“+” 作为一元操作符(正负号中的正号)

  • 作用：对于数值类型, 不会产生任何影响； 对于非数值 转换为数值；
  • 规则：
    • 基本数据类型 调用Number(), 函数进行转换;
    • 复杂数据类型 调用valueOf,或者toString(), 进行转换.

  1. + Null -> 0

  2. + Undefined -> NaN;

  3. + Number -> Number

  4. + String -> to Number(); – 空字符串 -> 0; 非空字符串 -> NaN

  5. + Boolean -> to Number(); – false -> 0; true -> 1

  6. + Object -> valueOf()|| toString();

“+” 作为二元操作符(加减乘除的+)

  • 作用: 两个操作数的计算.
  • 规则:
    1. 如果有一个操作数为String类型, 则转换为String类型, 然后进行字符串拼接操作.
    2. 如果一个操作数为Number 且另一个操作数不是String, Object， 则调用Number(), 转换为数字.
    3. 关于Infinity
    4. 关于0
    5. 一个操作数为Object, 另一个操作数为Boolean,Number, 则转换为字符串. null 和undefined 调用String();
    1. 一个操作数为String
    1. 一个操作数为Number
    1. Infinity(正无穷大)
    1. 关于0
    1. 一个操作数为{}(与操作数的顺序有关分成两种情况)
