‘JavaScript == 云算法’

‘==’ 比较两个操作数是否相等，当两边类型不同进行隐式类型转换.

  • 规则：
    • 一个操作数为boolean, 转换为数值进行比较.
    • 一个数值一个字符串, 转换为数值进行比较.
    • 一个对象另一个不是, 调用valueOf，转换为基本数据类型进行比较.
    • null == undefined // true; undefined 和null, 不进行转换。
    • 有一个操作数为NaN，结果NaN
    • 两个对象比较他们是否是同一个引用.

一个操作数为Boolean.

一个数值，一个字符串. 转换为数值进行比较.
